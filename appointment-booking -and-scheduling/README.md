# AI Appointment Booking & Scheduling Bot

## Overview
This project is an AI-powered appointment booking assistant built using n8n, Telegram, Groq AI, and Google Calendar.

The bot can chat with users on Telegram, understand appointment requests, check calendar availability, create events, and send confirmation messages automatically.

## Features

- AI-powered conversation
- Telegram bot integration
- Google Calendar integration
- Appointment scheduling
- Calendar availability check
- Memory support for conversations
- Automatic confirmation messages

## Tech Stack

- n8n
- Groq AI
- Telegram Bot API
- Google Calendar API

## Workflow

Telegram → AI Agent → Google Calendar → Telegram Reply

## How to Use

1. Create a Telegram bot using BotFather.
2. Connect Telegram credentials in n8n.
3. Connect Google Calendar.
4. Configure the Groq model.
5. Import the workflow.
6. Publish the workflow.
7. Start chatting with the Telegram bot.
